About the Project

Every year the student’s final year presents a project that is called the final year project there is no proper documentation for that data and is not available in one place so it’s hard to find the data if someone wants to see the work of previous student’s project and thesis so creating a proper website that contains all the data about the FYP and their thesis will be a great idea. The website will contain the FYP details for every year and all the information about the students their projects and the supervisors we want to search by student name or registration number or by the supervisor’s name that will display the desired list of the data which makes it easier and easy to access
